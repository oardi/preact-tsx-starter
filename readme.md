# Preact Typescript TSX Webpack Template

Preact Typescript Template with TSX Support, SCSS and Webpack 4.
Take a look into app.tsx and button-counter.tsx 

## Usage
* `npm install`
* `npm start`

## Build
* `npm run build`
* `npm run build:prod`

## Included
- preact
- bootstrap
- webpack 4
- typescript 3
- tsloader 5
